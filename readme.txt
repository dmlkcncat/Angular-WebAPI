Projenin backendi .Net Core Web API, frontendi ise Angular ile geliştirilmiştir. 
Veritabanı Microsoft SQL Server tercih edilmiştir.

Kodlar mevcut dosya yapısında frontend-backend olarak ayrı klasörlenmiştir.
SQL uzantılı scriptlerde CREATE TABLE scriptleri bulunmaktadır.

Backend WEB API kısmında inheritance, abstraction, interface gibi SOLID kavramlara 
dikkat edilmiştir. 

Front end kısmında Style sheet olarak bootstrap'ten yararlanılmıştır.

Front-end Angular kısmında node-modules (NPM paketlerinin bulunduğu klasör>300 mb) 
mail atma kolaylığı açısından silinmiştir.

https://github.com/dmlkcncat/Angular-WebAPI üzerinde proje paylaşılmıştır.
